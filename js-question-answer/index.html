Question & Answer

When was JavaScript developed?

JavaScript was developed by Brendan Eich at Netscape in 1995.

What does the acronym "DOM" stand for?

"DOM" stands for Document Object Model. It enables the structure of HTML or XML documents to be accessed and manipulated by programs.

What is a "Closure" and how does it work in JavaScript?

A closure is a JavaScript feature that allows a function to access variables from its outer scope. It occurs when a function is referenced by an inner function.

What does "NaN" mean in JavaScript?

"NaN" stands for "Not-a-Number." It represents the idea that a value is not a valid number or the result of an undefined numerical operation.

What is a "Callback function" in JavaScript and why is it used?

A callback function is a function passed as a parameter to another function, and it is invoked after the completion of a specific task. It is commonly used in asynchronous programming to trigger another operation when a task is completed.
